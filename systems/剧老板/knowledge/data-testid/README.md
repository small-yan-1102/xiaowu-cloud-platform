## 命名规范

**格式**：`{模块编号}_{场景编号}_{序号}`

| 部分 | 说明 | 示例 |
|------|------|------|
| 模块编号 | 两位数字，与功能模块对应 | `01`、`02` |
| 场景编号 | `layout`（布局）或 `S{N}-{M:02d}`（场景N第M步） | `layout`、`S1-01`、`S2-01` |
| 序号 | 两位数字，同一场景内从 01 递增 | `01`、`02`、`03` |
