## 1. 这个功能是干什么的

公司与分销商解约、版权纠纷等场景下，需要批量将 YouTube 上的视频删除或设为私享。**视频下架**模块就是管理这个流程的工具：

- **谁用**：运营/版权人员在 AMS 系统中创建和管理下架任务
- **做什么**：选择要下架的作品/视频 → 提交审核 → 系统自动调用 YouTube API 执行下架
- **结果在哪看**：AMS 详情页查看执行进度；分销商在剧老板系统查看已完成的下架记录

---

## 2. 30 秒了解核心流程

```
你创建任务单 → 系统获取视频ID → 有审核权限的人审核 → 系统自动排队执行下架 → 完成
```

用状态表示：

```
创建中 → 待审核 → 待处理 → 处理中 → 已完成
```

整个过程中你只需要做两件事：**创建** 和 **审核**。其余全部由系统自动完成。

## 3. 任务单状态一览

| 状态 | 颜色 | 含义 | 你能做什么 |
|------|------|------|-----------|
| **创建中** | 橙色 | 系统正在获取视频ID，请等待 | 等待（通常几秒~几分钟） |
| **创建失败** | 红色 | 视频ID获取失败 | 编辑后重新提交，或删除 |
| **待审核** | 橙色 | 等待有审核权限的人审核 | 有权限则审核 |
| **审核拒绝** | 红色 | 审核未通过（悬停可看原因） | 编辑后重新提交，或删除 |
| **待处理** | 橙色 | 已进入执行队列，等待额度分配 | 只能查看 |
| **处理中** | 橙色 | 正在调用 YouTube API 执行下架 | 只能查看 |
| **已完成** | 绿色 | 全部视频已处理完毕 | 查看详情、导出结果 |

> 如果已完成状态旁边有 ℹ️ 图标，说明部分视频下架失败，点击进详情页查看。

## 4. 操作指南：创建任务单

### 入口

AMS 系统 → 视频管理 → 视频下架 → 右上角【创建任务单】下拉菜单

有三种创建方式：

| 你的场景 | 选择方式 | 入口 |
|---------|---------|------|
| 知道作品名称，整部下架 | **按作品创建** | 视频下架列表 → 【创建任务单】→ 按作品创建 |
| 知道具体视频ID | **按视频ID创建** | 视频下架列表 → 【创建任务单】→ 按视频ID创建 |
| 分销商解约，需要同时解约+下架 | **从解约单自动创建** | 内容解约单 → 创建解约单时自动生成 |

---

### 方式一：按作品创建（最常用）

**第 1 步：填写基础信息**

| 字段 | 必填 | 说明 |
|------|------|------|
| 下架原因 | 是 | 下拉选择（6个选项） |
| 下架说明 | 否 | 补充说明，最多100字 |
| 处理方式 | 是 | 视频删除（不可恢复）/ 视频私享（可恢复） |
| 截止执行日期 | 是 | 今天 ~ 今天+6天 |

**第 2 步：添加作品**

两种方式，可混合使用：

**方式 A — 手动选择**：
1. 点击【+ 新增作品】
2. 在弹窗中搜索作品名称
3. 勾选目标作品（支持多选、跨页选择）
4. 点击【确认】

**方式 B — 批量导入**：
1. 点击【批量导入】
2. 下载模板，填入作品名称、CP名称、海外频道ID
3. 上传文件（xls/xlsx，≤5MB，≤500条）
4. 系统自动校验，通过则导入

> 注意：第二次导入会**覆盖**第一次的数据，不是追加。手动选择的作品不受影响。

**第 3 步：选择海外频道（可选）**

点击作品行中的"海外频道ID"列 → 弹窗中勾选要下架的频道 → 保存。

不选则默认该作品下所有频道的视频都下架。

**第 4 步：提交**

点击【提交审批】→ 系统校验 → 提交成功后返回列表页，任务单状态为"创建中"。

---

### 方式二：按视频ID创建

**第 1 步：填写归属信息**（必须先填，否则无法导入）

| 字段 | 说明 |
|------|------|
| 任务来源 | 分销商 / 运营团队 |
| 分销商名称 | 来源=分销商时必填 |
| 运营团队 | 来源=运营团队时必填 |

> 导入数据后归属信息会**锁定**。要改归属需先清空导入数据。

**第 2 步：填写基础信息**（同上）

**第 3 步：批量导入视频ID**

1. 点击【批量导入】
2. 下载模板，填入视频ID（11位字符）、作品名称、CP名称、海外频道ID
3. 上传文件（≤5MB，≤1000条）
4. 系统校验通过后，自动按作品分组展示

**第 4 步：提交**（同上）

---

### 方式三：从解约单自动创建

当分销商解约需要同时下架视频时，不需要手动创建下架任务单——创建解约单时系统会自动生成。

**入口**：AMS 系统 → 内容资产 → 内容解约单 → 【创建解约单】下拉菜单

创建解约单有 **2 种模式**：

| 模式 | 适用场景 | 添加作品方式 |
|------|---------|------------|
| **按分销商创建** | 已知哪个分销商要解约 | 先选分销商，再从该分销商名下选作品（下拉多选） |
| **按作品创建** | 已知哪些作品要解约 | 先选作品（左右双栏表格），再选每个作品的解约分销商 |

**操作步骤**：

**第 1 步：选择创建模式**（按分销商 / 按作品）

**第 2 步：填写基本信息**

| 字段 | 必填 | 说明 |
|------|------|------|
| 分销商名称 | 是（按分销商模式） | 下拉选择，按作品模式下在后续步骤中选择 |
| 解约类型 | 是 | 下拉选择，3 个选项（见下方联动规则） |
| 解约原因 | 是 | 文本输入 |
| 解约日期 | 是 | 日期选择器 |

**第 3 步：解约类型选择 — 决定是否联动视频下架**

这是最关键的一步，不同解约类型会导致完全不同的后续表单：

| 解约类型 | "视频是否下架"字段 | 下架信息区块 | 提交后是否创建下架任务单 |
|---------|-----------------|------------|---------------------|
| **版权方解约** | 显示，默认"是"，**不可修改** | 显示，下架原因**自动填充且锁定** | 是 |
| **分销商解约** | 显示，默认"是"，**不可修改** | 显示，下架原因**自动填充且锁定** | 是 |
| **双方协商一致** | **整个字段隐藏** | **整个区块隐藏** | 否（仅创建解约单） |

当解约类型为版权方解约或分销商解约时，下架信息区块中的字段：

| 字段 | 状态 | 说明 |
|------|------|------|
| 视频是否下架 | 锁定为"是" | 不可修改 |
| 下架原因 | 自动填充，不可编辑 | 根据解约类型自动匹配（版权方解约→"上游版权解约"；分销商解约→"分销商解约"） |
| 下架说明 | 可编辑 | 选填，补充说明 |
| 处理方式 | 默认选中，禁用状态 | 默认"视频删除"，不可修改 |
| 截止执行日期 | **需手动选择** | 必填；必须先选解约日期才能选此字段；范围：max(今天, 解约日期) ~ +6天 |

> 简单说：选了版权方解约或分销商解约后，大部分下架字段自动填好了，你只需要选一个截止日期。注意截止日期**依赖解约日期**，起点不一定是今天，而是今天和解约日期中较晚的那个。

**第 4 步：添加作品**

- 按分销商模式：点击【添加作品】→ 下拉多选该分销商名下的作品 → 确定
- 按作品模式：点击【添加作品】→ 左右双栏搜索选择作品 → 确定 → 再为每个作品选择解约分销商

**第 5 步：提交**

点击【解约】→ 如果视频下架=是，弹出确认弹窗提示"将自动创建视频下架任务单" → 确认后提交

**提交后自动发生的事**：

按分销商创建和按作品创建的提交后行为有差异：

**按分销商创建**：

```
提交解约单（已选 1 个分销商 + N 个作品）
    │
    ▼
生成 1 张解约单（该分销商）
    │
    ▼
若视频下架=是 → 自动创建 1 个视频下架任务单
    │
    ▼
任务单进入"创建中"→ 后续流程与手动创建完全相同
```

**按作品创建**：

```
提交解约单（已选 M 个作品，每个作品选了不同的解约分销商）
    │
    ▼
系统自动按分销商分组拆分 → 每个分销商生成一张独立的解约单（共 N 张）
    │
    ▼
若视频下架=是 → 所有作品聚合，只创建 1 个视频下架任务单
    │
    ▼
任务单进入"创建中"→ 后续流程与手动创建完全相同
```

> 关键区别：按分销商模式是 1 对 1（1 个分销商 → 1 张解约单）；按作品模式可能 1 次操作产生多张解约单（按分销商自动拆分），但下架任务单始终只有 1 个。

**注意事项**：

| 要点 | 说明 |
|------|------|
| 解约单和任务单独立 | 解约单立即生效（无审批流程）；任务单仍需审核通过后才执行下架 |
| 一次创建，多张解约单 | 按分销商模式：1个分销商=1张解约单；按作品模式：系统自动按分销商分组拆分，只生成1个下架任务单 |
| 不可撤销 | 解约单创建后不可取消/作废 |
| 视频ID来源 | 系统通过"作品名称+分销商名称+海外频道ID"自动匹配剧老板的视频登记数据 |
| 详情页可追溯 | 自动创建的任务单在详情页"创建来源"字段显示"解约单创建"（手动创建的显示"页面创建"） |

---

### 创建后会发生什么

```
提交成功
    │
    ▼
系统异步获取视频ID（"创建中"状态）
    │
    ├── 至少1个作品有可执行视频 → 创建成功 → 状态变为"待审核"
    │
    └── 所有作品都没有可执行视频 → "创建失败"
        （可编辑后重试，或删除）
```

## 5. 操作指南：审核任务单

### 谁能审核

在权限中心配置了"审核"按钮权限的用户。没有权限的用户看不到审核按钮。

### 怎么审核

1. 在列表页找到"待审核"状态的任务单
2. 点击操作列的【审核】按钮
3. 选择审核结果：

| 选择 | 审核意见 | 结果 |
|------|---------|------|
| **通过** | 选填（≤200字） | 任务单进入"待处理"，开始排队执行 |
| **不通过** | **必填**（≤200字） | 任务单变为"审核拒绝" |

4. 点击【确认】

> 审核拒绝后，创建者可以编辑任务单修改后重新提交。

## 6. 操作指南：查看详情与导出

### 查看详情

点击列表页的任务单编号链接 → 打开详情抽屉。

详情页展示三块信息：

| 区域 | 展示内容 |
|------|---------|
| 基础信息 | 编号、创建人、下架原因、处理方式、截止日期、审核意见 |
| 整体完成进度 | 提交总数、已成功、处理失败、排队中、终止、完成百分比 |
| 作品处理进度 | 按作品分组展开，每个视频的处理状态和失败原因 |

### 导出

| 按钮 | 出现条件 | 作用 |
|------|---------|------|
| 导出全部文件 | 任务单已完成 | 导出所有视频的处理结果 |
| 导出失败文件 | 已完成且有失败视频 | 仅导出失败视频，便于排查 |

> 单次导出上限 1000 条。超过时请用搜索条件缩小范围后再导出。

## 7. 操作指南：编辑与删除

| 操作 | 可操作的状态 | 说明 |
|------|------------|------|
| **编辑** | 创建失败、审核拒绝 | 修改后重新提交，会重新走"创建中→待审核"流程 |
| **删除** | 创建失败、审核拒绝 | 物理删除，不可恢复，需二次确认 |

> 待审核/待处理/处理中/已完成 状态的任务单**不可编辑和删除**。

## 8. 额度与排队机制

### 你需要知道的

- 系统每天有 **5000 条视频** 的下架额度（仅"视频删除"消耗，"视频私享"不消耗）
- 额度每天 16:00 更新
- 多个任务单按优先级排队执行，你不需要手动干预

### 排队优先级（系统自动）

```
① 截止日期是今天的任务优先
② 截止日期更近的优先
③ 下架原因优先级：临时版权纠纷 > 调整上线时间 > 单作品解约 > 上游版权解约 > 分销商解约 > 其他
④ 审核通过时间更早的优先
```

### 额度不足时

- 系统会执行到额度用完为止，剩余视频自动排到次日
- 如果截止日期到了还没执行完，系统会发钉钉告警通知相关人员

## 9. 常见问题

### Q：创建失败了怎么办？

**原因**：系统没有找到任何可执行的视频ID（可能是作品没有绑定视频，或所有视频已被其他任务单下架）。

**处理**：点击【编辑】检查作品和频道选择是否正确，修改后重新提交。

---

### Q：部分作品创建失败，整个任务单算失败吗？

**不算**。只要有至少1个作品获取到了视频ID，任务单就创建成功。失败的作品会在详情页标注。

---

### Q：导入/提交时提示"视频ID已存在于其他任务单"怎么办？

系统会检查每个视频ID是否已在其他**进行中**的任务单里。具体规则：

| 视频在其他任务单的状态 | 能否再次导入 | 说明 |
|---------------------|------------|------|
| 创建中 / 待审核 / 待处理 / 处理中 | **不能** | 被拦截，提示"该视频已在执行中的任务单中" |
| 创建失败 / 审核拒绝 | **能** | 前次任务单已终止，不阻塞 |
| 已完成 - 下架**成功** | **不能** | 视频已下架，无需重复操作 |
| 已完成 - 下架**失败** | **能** | 允许重新导入执行 |

**怎么处理**：

1. 先确认该视频在哪个任务单里——在视频下架列表用视频ID搜索（精确匹配，会搜到所有包含该视频的任务单）
2. 如果前次任务单还在执行中 → 等待完成后再操作
3. 如果前次任务单已完成但视频下架失败 → 可以直接导入到新任务单重试
4. 如果前次是"视频删除"成功 → 无法再次操作（已永久删除）
5. 如果前次是"视频私享"成功，现在要改为删除 → 可以创建新任务单选择"视频删除"

### Q：同一个视频先私享后删除，或先删除后私享，支持吗？

| 场景 | 是否支持 | 说明 |
|------|---------|------|
| 先**私享** → 再**删除** | 支持 | 创建新任务单选"视频删除"即可 |
| 先**删除** → 再**私享** | 不支持 | 视频已永久删除，无法再操作 |
| 先**删除** → 再**删除** | 不支持 | 系统过滤，提示"视频已删除" |

---

### Q："视频删除"和"视频私享"有什么区别？

| | 视频删除 | 视频私享 |
|---|---------|---------|
| YouTube 效果 | 视频永久删除 | 视频设为私享（仅频道主可见） |
| 是否可恢复 | 不可恢复 | 可恢复 |
| 消耗额度 | 是（每条1额度） | 不消耗 |
| 排队 | 需要排队 | 不排队，直接执行 |

---

### Q：已经删除的视频能改成私享吗？

**不能**。删除是不可逆的。但**私享的视频可以再次删除**（创建新任务单选择"视频删除"）。

---

### Q：截止日期过了还没执行完怎么办？

系统会继续执行，不会自动取消。同时会通过钉钉告警通知相关人员（17:00 触发）。

---

### Q：视频下架失败了怎么处理？

在详情页查看失败原因：

| 失败原因 | 含义 | 处理方式 |
|---------|------|---------|
| 视频不存在 | YouTube 上找不到该视频 | 确认视频ID是否正确 |
| 无权限 | 频道没有操作权限 | 联系运营确认频道授权 |
| 视频已删除 | 之前已经被删除了 | 无需处理 |
| 网络超时/限流 | 临时性错误 | 系统会自动重试（最多3次） |

---

### Q：在剧老板系统能看到什么？

分销商在剧老板的"视频管理 → 视频下架列表"中，只能看到**已完成**的任务中归属自己的视频。待处理/处理中的任务不会展示。数据通常在任务完成后 30 分钟内同步。
