## 2. 创建解约单（抽屉）

> 抽屉宽度：950px，标题：「内容解约单」
> 分两种创建模式，由 `type` 参数控制

### 2.1 两种模式对比

| 维度 | 按分销商创建（`type=team`） | 按作品创建（`type=work`） |
|------|--------------------------|------------------------|
| 入口提示 | 先选分销商，再选该分销商下的作品 | 直接选作品，系统自动按分销商生成多张解约单 |
| 分销商名称字段 | ✅ 显示（必填） | ❌ 隐藏（`v-if="!isWork"`） |
| 添加作品弹窗 | 加载该分销商下待解约作品（下拉多选） | 全局作品搜索表格（左右双栏选择） |
| 解约分销商名称列 | ❌ 不显示 | ✅ 显示（可选多个分销商） |
| 提交接口 | `POST /compositionTerminate/create` | `POST /compositionTerminate/batchCreate` |
| 生成结果 | 1 张解约单 | 按分销商自动分组，生成多张解约单 |

### 2.2 基本信息区字段

| 字段名 | 必填 | 组件类型 | 校验规则 | 按模式显示 |
|------|:----:|---------|---------|-----------|
| 分销商名称 | ✅ | 下拉搜索（接口拉取） | 必选 | 仅「按分销商创建」 |
| 解约类型 | ✅ | 下拉 | 必选；枚举来源字典表 | 两种模式均显示 |
| 解约原因 | ✅ | 输入框 | 必填；最大长度 200 字符 | 两种模式均显示 |
| 解约日期 | ✅ | 日期选择器 | 必选；单日期 | 两种模式均显示 |

**解约类型枚举值**（字典表 `compositionTerminationType`）：

| 枚举值 |
|------|
| 双方协商一致 |
| 分销商解约 |
| 上游版权解约 |

### 2.3 解约信息区（作品表格）

**表格列按模式差异**：

| 列名 | 按分销商创建 | 按作品创建 |
|------|:-----------:|:---------:|
| 作品名称 | ✅ | ✅ |
| 作品链接 | ✅ | ✅ |
| 作品规格 | ✅ | ❌ |
| 运作套餐 | ✅ | ❌ |
| 运作语种 | ✅ | ❌ |
| 分配日期 | ✅ | ❌ |
| 解约分销商名称 | ❌ | ✅（点击可选多个分销商） |

**行内操作**：点击行首 `-` 图标，删除该作品行。

### 2.4 解约信息区操作入口

| 操作 | 行为 | 前置校验 |
|------|------|---------|
| 添加作品 | **按分销商**：弹出该分销商下待解约作品列表（下拉多选，支持全选）<br>**按作品**：弹出全局作品搜索表格（左侧搜索+右侧已选，按作品名模糊搜索） | 按分销商模式下：未选分销商时提示「请选择分销商」 |
| 批量导入 | 触发 Excel 批量导入；单次限制 500 条，文件不超过 5M | 按分销商模式下：未选分销商时提示「请选择分销商」 |
| 解约分销商名称（按作品模式） | 点击单元格打开 TeamDialog，展示该作品已分配的分销商列表 | 分销商不能为空（提交时校验） |

### 2.5 批量导入 Excel 模板

> 字段定义来源：后端 `CompositionTerminateImport.java`（EasyExcel 注解）

| 列序号 | 列名 | 是否必填 | 校验规则 |
|-------|------|:-------:|---------|
| 0 | `* 作品名` | ✅ | 必填；只能填写已分配给该分销商且未解约的作品名称；文件内不能重复；匹配不到已分配作品报错 |

> 其余字段（作品链接、规格、套餐、语种等）均为 `@ExcelIgnore`，不在模板中出现，由后端根据作品名自动匹配填充。

**后端校验规则（importAdd 接口）**：

| 校验项 | 规则 | 错误提示来源 |
|-------|------|------------|
| 文件内重复 | 同名作品在导入文件中出现 ≥ 2 次 | `COMPOSITION_NAME_REPEAT` |
| 作品不存在 | 作品名在已分配列表中匹配不到 | `COMPOSITION_NAME_NOT_ALLOCATED` |
| 同名多条 | 同分销商下同名作品匹配到多条记录 | `COMPOSITION_NAME_DB_MULTI` |

### 2.6 选择解约分销商（TeamDialog）

> 触发时机：按作品创建模式下，点击解约分销商名称单元格
> 标题：「选择解约分销商」，宽度：750px

| 元素 | 说明 |
|------|------|
| 搜索框 | 按分销商名称过滤列表（前端本地过滤，不发请求） |
| 列表字段 | 分销商名称 / 套餐 / 语种 / 交接日期 |
| 选择方式 | **多选**（checkbox，无数量上限） |
| 提示文案 | 「仅展示已分配给分销商且未解约的作品」 |
| 提交校验 | 至少选 1 个分销商，否则提示「解约分销商不能为空」 |

### 2.7 提交逻辑

| 校验项 | 规则 |
|------|------|
| 表单必填 | 分销商名称（按分销商模式）/ 解约类型 / 解约原因 / 解约日期 全部通过 |
| 作品数量 | 至少添加 1 个作品，否则提示「最少添加一个作品」 |
| 作品上限 | 最多 500 个（超出自动截取前 500 条，并提示「最多添加500个作品」） |

**提交成功**：提示「创建解约单成功」→ 刷新列表 → 关闭抽屉

## 4. 接口清单

| 接口名称 | Method | URL | 说明 |
|---------|--------|-----|------|
| 解约单列表（分页） | POST | `/compositionTerminate/pageList` | 搜索条件：分销商名称 / 解约作品名 / 解约日期范围 |
| 创建解约单（按分销商） | POST | `/compositionTerminate/create` | 单张创建 |
| 批量创建解约单（按作品） | POST | `/compositionTerminate/batchCreate` | 系统按分销商自动分组，生成多张 |
| 待解约作品列表 | POST | `/compositionTerminate/compositionList` | 按 `teamId` 筛选该分销商下已分配且未解约的作品 |
| 解约单详情 | POST | `/compositionTerminate/detail/{id}` | 返回解约信息 + 作品明细 |
| 作品简单列表（按作品创建时搜索） | POST | `/compositionTerminate/compositionSimpleList` | 按作品名模糊搜索（`compositionNameLike`） |
| 解约批量导入 | POST | `/compositionTerminate/importAdd` | 参数含 `teamId`（按分销商模式）；Excel 模板仅「* 作品名」一列 |
| 分销商列表（创建时下拉） | POST | `/team/queryTeamList` | 复用资产账户频道/主页模块接口 |
