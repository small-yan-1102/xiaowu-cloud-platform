## 命名规范

结算系统采用 **语义化命名**，与 AMS/剧老板的数字编号体系不同。

### 基本格式

```
[模块]_[页面/组件]_[元素类型]_[功能/标识]
```

### 命名组成部分

| 组成部分 | 规则 | 示例 |
|---------|------|------|
| **模块** | 系统模块缩写，小写 | `yt` / `fb` / `invoice` / `data` / `config` |
| **页面/组件** | 功能页面或组件名，小写 | `settlement` / `vat` / `proforma` / `dashboard` |
| **元素类型** | 元素类型标识 | `tab` / `btn` / `input` / `table` / `modal` |
| **功能/标识** | 具体功能描述或唯一标识 | `pending` / `batch_audit` / `row_001` |

### 元素类型缩写

| 类型 | 缩写 | 说明 |
|------|------|------|
| Tab栏 | `tab` | 状态Tab切换 |
| 按钮 | `btn` | 所有按钮 |
| 输入框 | `input` | 文本/数字输入 |
| 下拉选择 | `select` | 下拉框/选择器 |
| 表格 | `table` | 数据表格 |
| 表格行 | `row` | 表格行（带索引） |
| 表格列 | `col` | 表格列 |
| 弹窗 | `modal` | 弹窗/模态框 |
| 日期选择 | `datepicker` | 日期范围选择 |
| 复选框 | `checkbox` | 多选框 |
| 单选框 | `radio` | 单选框 |
| 链接 | `link` | 可点击链接 |
| 图标 | `icon` | 功能图标 |
| 标签 | `tag` | 状态标签 |
| 统计卡片 | `stat` | 统计数字卡片 |
| 菜单 | `menu` | 下拉菜单 |
| 步骤条 | `step` | 审核流程步骤 |
| 分页 | `pagination` | 分页组件 |
| 提示 | `tooltip` | 提示信息 |

> **例外**：模块 11（逾期结算处理）使用数字编号体系 `{模块编号}_{场景编号}_{序号}`，与 AMS 规范一致。

## 表格行处理建议

| 方案 | 适用场景 | 说明 |
|------|---------|------|
| 方案A：行索引 `row_001` | 行数固定、顺序不变 | 简单直观 |
| 方案B：`data-row-id` 属性 | 动态行数、行顺序变化 | 基于业务ID，稳定可靠 |

> **推荐**：财务结算系统数据量大且经常变化，建议使用 **方案B（data-row-id）**。
