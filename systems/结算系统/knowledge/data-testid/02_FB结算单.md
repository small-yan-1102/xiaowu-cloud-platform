# FB结算单管理 (`fb_settlement`)

> 模块编号：02 | 元素数量：~70 | 主要页面：列表、详情、审核、生成
>
> FB结算单与YT结构类似，前缀改为 `fb_settlement`。未单独列出的元素参考 [01_YT结算单.md](01_YT结算单.md) 中对应部分，将前缀 `yt_settlement` 替换为 `fb_settlement`。

---

## 1. Tab栏

```yaml
fb_settlement_tab_all:                      # 全部
fb_settlement_tab_pending_customer:         # 待客服审核
fb_settlement_tab_pending_finance:          # 待财务审核
fb_settlement_tab_pending_review:           # 待客服二审
fb_settlement_tab_pushed:                   # 已推送
fb_settlement_tab_hold:                     # 暂不结算
```

## 2. FB特有功能

```yaml
fb_settlement_btn_export_with_product:      # 导出（含Product版）
fb_settlement_btn_export_without_product:   # 导出（不含Product版）
```

> 其余筛选条件、操作按钮、统计区、表格、审核弹窗、详情抽屉等结构与 YT 结算单一致，前缀为 `fb_settlement`。
