# 增值税发票 (`invoice_vat`)

---

## 1. Tab栏

```yaml
invoice_vat_tab_pending_upload:             # 待上传
invoice_vat_tab_pending_audit:              # 待审核
invoice_vat_tab_rejected:                   # 审核驳回
invoice_vat_tab_audited:                    # 已审核
invoice_vat_tab_remittance:                 # 已汇款
```

```yaml

```
