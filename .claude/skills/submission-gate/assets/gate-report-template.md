
## 门禁状态符号说明

| 符号 | 含义 |
|------|------|
| ✅ 通过 | 该模块无阻塞，可进入下游测试设计 |
| ❌ 阻塞 | 该模块存在阻塞项，禁止进入下游测试设计 |
| 🟢 可生成E2E用例 | 前端+后端文档均审阅完成，无阻塞性问题 |
| 🟡 技术文档不完整 | 仅后端/接口文档审阅完成，前端文档待补充 |
| 🔴 建议等待 | 存在未解决的阻塞性问题（B级），或核心文档未提供 |
