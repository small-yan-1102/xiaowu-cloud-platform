## 二、间距规范

| 位置 | 间距值 | 说明 |
|------|--------|------|
| 区块之间 | 14px - 16px | 区块间距 |
| 区块内边距 | 12px - 16px | 内边距 |
| 表单项间距 | 16px - 24px | 表单项间 gap |
| 按钮间距 | 8px | 按钮间 gap |
| 列表项间距 | 8px - 12px | 列表项间距 |

---

## 三、表单布局规范

### 搜索表单
- 每行 3-4 个字段
- 标签宽度自适应
- 输入框宽度 200px - 300px

### 普通表单
- 标签宽度 100px（标签在左布局）
- 单行 2-3 列（grid 布局）

## 六、按钮规范

| 类型 | 使用场景 |
|------|---------|
| 主按钮 | 提交、确认、主要操作 |
| 次按钮 | 取消、重置、次要操作 |
| 文字按钮 | 链接式操作 |
| 危险按钮 | 删除、危险操作 |

### 按钮布局
- 操作区：左次要、右主要
- 弹窗/抽屉底部：取消在左、确认在右

---

## 七、通用审美准则

1. **对齐原则**：标签右对齐、数值右对齐、文本左对齐、操作居中
2. **对比原则**：标题 > 内容 > 辅助信息（字号递减）
3. **一致性**：相同功能相同图标、相同操作相同按钮样式
4. **留白**：区块间留白 > 区块内留白
5. **视觉层次**：页面标题 → 区块标题 → 字段标签 → 字段值
