## 5. 数据产出计划

本套件的数据全部依赖AMS视频下架核心流程的执行产出：

| 所需数据 | 产出来源 | 产出步骤 |
|---------|---------|---------|
| 已私享视频ID | VT-006 视频私享任务→执行→已完成 | 从已完成详情中获取已私享的视频ID |
| 已删除视频ID | VT-005 视频删除任务→执行→已完成 | 从已完成详情中获取已删除的视频ID |
| 失败视频 | 任务执行后自然产出（部分视频可能失败） | 从详情中获取失败视频ID和失败原因 |
| 全部失败任务 | 需特殊构造：使用全部无效的视频ID | 故意导入不存在的视频ID |
| 重试中视频 | 执行过程中的临时状态 | 需在执行过程中抓拍 |

> **执行策略**：通用规范套件应安排在AMS视频下架核心用例（suite_ams_video_takedown.md Round 1-10）完成之后执行。
