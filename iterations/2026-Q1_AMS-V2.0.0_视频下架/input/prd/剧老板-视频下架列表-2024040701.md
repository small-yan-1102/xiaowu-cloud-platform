# 剧老板-视频下架列表

> **说明：** 本文档包含剧老板端视频下架功能的完整需求描述。

---

## 一、视频下架列表页

### 1.1 页面概述

**数据权限：** 分配该作品的运营人员及拥有该运营人员管理范围的人员可见

**展示维度：** 按作品+频道维度展示，仅展示已完成状态的数据

**展示规则：**
- 一个作品在一个频道下的下架任务为一条数据
- 同一作品在多个频道下架时，列表展示多条数据（每个频道一条）
- 示例：作品"恶魔之吻"在3个频道下架，列表展示3条数据

| 作品名称 | 频道名称/ID | 下架视频数 |
|---------|-----------|-----------|
| 恶魔之吻 | Nhiệp Chính Drama / UCxxx | 38 |
| 恶魔之吻 | TM - Drama Studio / UCyyy | 15 |
| 恶魔之吻 | VietDuc Network / UCzzz | 22 |

**数据推送规则：**

| 项目 | 说明 |
|-----|------|
| 推送触发时机 | 任务单状态 = 已完成时（包含"全部失败"场景）；**仅在任务单完成时推送一次（最终结果同步）** |
| 推送匹配规则 | 依据海外频道分配记录（`team_operate_record`表）匹配分销商：视频发布日期在分配期间内（`delivery_date` ≤ 发布日期 ≤ `termination_date`），推送给该期间的分销商 |
| 数据隔离规则 | 视频发布日期必须在分销商分配期间内，才推送给该分销商；**无匹配分销商的视频不推送** |
| 多匹配处理 | 如匹配到多条分配记录，取`created_at`最新的记录推送，并记录诊断日志 |
| **重复数据覆盖规则** | **列表按【作品+海外频道】唯一维度展示；同一作品+海外频道的多次推送，后覆盖前；视频明细按【视频ID】唯一维度展示，相同视频ID以最新状态为准** |
| 推送维度 | 作品+海外频道维度 |
| 推送目标 | 对应剧老板账号 |
| 视频状态映射 | AMS端"已完成"→剧老板"成功"；AMS端"处理失败"→剧老板"失败" |
| 完成时间 | 按视频级别推送（每个视频有自己的完成时间） |
| 同步失败处理 | **技术确认项**：RocketMQ消息发送失败时的重试和补偿机制由技术设计 |

**分销商匹配逻辑：**

```sql
SELECT * FROM team_operate_record 
WHERE channel_id = '视频所属海外频道ID'
AND delivery_date <= '视频发布日期'
AND (termination_date IS NULL OR termination_date >= '视频发布日期')
ORDER BY created_at DESC
LIMIT 1
```

**分配期间定义：**
- 分配期间 = [`delivery_date`, `termination_date`]
- 如果 `bind_status = 0`（正常）：`termination_date = 当前日期`
- 如果 `bind_status = 2`（解约）：`termination_date = 解约日期`

**RocketMQ消息体字段：**

| 字段名 | 类型 | 说明 |
|-------|------|------|
| taskNo | string | 任务单编号 |
| workName | string | 作品名称 |
| channelId | string | 海外频道ID |
| channelName | string | 海外频道名 |
| videoId | string | 视频ID |
| videoTitle | string | 视频标题 |
| **publishDate** | datetime | **视频发布日期，用于分销商数据隔离判定** |
| takedownStatus | string | 下架状态：成功/失败 |
| completeTime | datetime | 视频下架完成时间 |
| failReason | string | 失败原因（失败时必填） |

### 1.2 搜索条件
| 字段名称 | 输入类型 | 查询方式 |
|---------|---------|---------|
| 作品名称 | 文本输入 | 模糊查询 |
| 频道名称 | 文本输入 | 模糊查询 |
| 频道ID | 文本输入 | 精准查询 |
| 视频ID | 文本输入 | 精准查询 |
| 下架原因 | 下拉选择 | 全部、临时版权纠纷、调整上线时间、单作品解约、上游版权解约、分销商解约、其他原因 |

**交互规则：**
- 支持检索条件交叉查询
- 支持点击「重置」按钮清空所有检索条件，恢复默认展示状态

### 1.3 列表字段
| 字段名称 | 说明 |
|---------|------|
| 作品名称 | 作品名称 |
| 频道名称/ID | 频道名称和ID |
| 下架视频数 | 该作品在该频道下的视频数量；鼠标悬停可查看视频ID明细 |
| 下架原因 | 任务单的下架原因，数据来源：推送的作品关联的任务单信息 |
| 运营人员 | 该作品的运营负责人，数据来源：剧老板系统中该作品的运营负责人 |
| 完成时间 | 下架完成时间 |
| 操作 | 查看详情 |

## 二、进度查看页

**页面说明：**
- 剧老板进度查看页为**单作品维度**展示（区别于资产管理系统的多作品维度）
- 展示当前剧老板有权限查看的单个作品的视频下架进度

**展示内容：**
- 作品基本信息
- 视频明细列表

**视频明细列表字段：**

| 字段 | 说明 |
|-----|------|
| 序号 | 自增序号 |
| 视频ID | 视频唯一标识 |
| 视频标题 | 视频标题 |
| 频道名称/ID | 频道名称和ID |
| 下架状态 | 成功/失败 |
| 完成时间 | 下架完成时间 |
| 失败原因 | 失败时显示具体原因 |

**注：** 不展示【整体完成进度】模块
