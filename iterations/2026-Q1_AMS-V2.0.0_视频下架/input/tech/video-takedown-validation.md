# 视频下架任务校验逻辑清单

## 按视频ID导入

### 导入阶段校验

| # | 校验项 | 说明 |
|---|--------|------|
| 1 | 视频ID必填 | 空则报错 |
| 2 | 视频ID格式 | 长度必须为 11 位 |
| 3 | 作品名称必填 | 空则报错 |
| 4 | 海外频道ID必填 | 空则报错 |
| 5 | 视频ID批内重复 | 同批次内重复的所有行全部拦截（含第一条） |
| 6 | 视频ID在执行中任务已存在 | 查执行中任务中已存在则拦截 |
| 7 | 作品名称系统存在性 | 系统中不存在则报错 |
| 8 | CP名称匹配 | 有值时校验与作品是否匹配 |
| 9 | 海外频道ID系统存在性 | 不存在则报错 |
| - | 文件级限制 | 大小 ≤5M、格式 xlsx/xls、单次 ≤1000 条 |

### 提交阶段校验（异步执行，结果写入 detail）

| # | 校验项 | 失败原因 |
|---|--------|----------|
| 1 | 作品+CP 系统存在性 | 作品不存在 |
| 2 | 发布通道存在性及权限 | 无权限或未找到发布通道 |
| 2a | — 分销商权限 | taskSource=DISTRIBUTOR 时校验 composition_allocate 分配权 |
| 2b | — 运营团队权限 | taskSource=OPERATION_TEAM 时校验 sign_channel.operation_team |
| 3 | 视频已有执行中任务 | 视频已有执行中任务 |
| 4 | 视频已通过删除任务下架 | 视频已删除，无法再处理 |
| 5 | 视频已通过转私任务处理 | 视频已转私，无法再处理（仅处理方式=转私时校验） |

## 按作品导入

### 导入阶段校验

| # | 校验项 | 说明 |
|---|--------|------|
| 1 | 作品名称必填 | 空则报错 |
| 2 | 批内重复 | 作品名+CP名+海外频道ID 组合重复的所有行全部拦截（含第一条） |
| 3 | 作品名称系统存在性 | 不存在则报错 |
| 4 | CP名称匹配 | 有值时校验与作品是否匹配 |
| 5 | 作品+CP 唯一性 | 有CP时系统中该组合只能有一条记录 |
| 6 | CP为空时作品唯一性 | 无CP时系统中该作品只能有一条有效记录 |
| 7 | 海外频道ID格式 | 有值时：UC 开头、24位；支持多个（顿号/中英文逗号分隔） |
| 8 | 海外频道ID系统存在性 | 不存在时提示具体不存在的 ID |
| - | 文件级限制 | 大小 ≤5M、格式 xlsx/xls、单次 ≤500 条 |

### 提交阶段校验（异步执行，结果写入 detail）

| # | 校验项 | 失败原因 |
|---|--------|----------|
| 1 | 作品+CP 系统存在性 | 作品不存在 |
| 2 | 作品绑定海外频道 | 作品无绑定海外频道（未指定频道时自动查绑定频道） |
| 3 | 发布通道存在性 | 未找到发布通道（含合同生效状态校验） |
| 4 | 发布通道已配置 pipelineId | 发布通道未配置pipelineId |
| 5 | 视频已有执行中任务 | 视频已有执行中任务（批量检查） |
| 6 | 视频已通过删除任务下架 | 视频已删除，无法再处理（批量检查） |
| 7 | 视频已通过转私任务处理 | 视频已转私，无法再处理（仅处理方式=转私时校验，批量检查） |

---

## 通用说明

- 提交阶段异步校验失败 → detail `videoStatus = FAILED`，不重试
- 网络/系统异常 → detail `videoStatus = EXPAND_FAILED`，等待补偿任务重试
- 全部 detail 均为 FAILED → 任务状态变为 `CREATE_FAILED`
- 截止日期（当天~当天+6天）在提交接口入口同步校验
