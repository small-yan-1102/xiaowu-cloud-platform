### Quest 1: 视频下架列表页 - 列表展示与筛选

**产品原文**：
> 展示所有视频下架任务单，支持多维度筛选和查看。任务单编号生成规则：格式为V + YYMMDD + 四位数字，每日0点自动重置四位数字序列，从0001开始递增。
>
> **搜索条件**：作品名称（支持模糊搜索）、CP名称（支持模糊搜索）、处理方式（全部、视频删除、视频私享）、下架原因（全部、上游版权解约、分销商解约、单作品解约、临时版权纠纷、调整上线时间、其他原因）、截止执行日期（时间范围搜索）、创建来源（全部、页面创建、解约单创建）、视频ID（精确搜索）、海外频道ID（精准搜索）、分销商名称（模糊搜索）、处理状态（仅展示在Tab-全部中）。
>
> **一级列表字段**：任务单编号（点击可查看详情；左侧有展开箭头）、下架作品数、处理方式、下架原因、下架说明（超出长度截断显示，鼠标悬停展示全部）、截止执行日期、处理状态、创建来源、创建人、创建时间、完成时间（仅已完成状态）、操作。
>
> **二级列表字段**（展开后）：序号、作品名称、CP名称、海外频道名/ID（合并展示）、分销商名称（多个时用逗号隔开）。

**原型图解读**：
参考原型图`视频下架原型_v2.html`，列表页采用表格布局，支持展开/收起二级列表。一级列表展示任务单维度信息，二级列表展示作品-CP-频道绑定关系。搜索区域位于列表上方，采用多行筛选条件布局。

**需求解读**：

- 列表页是视频下架功能的入口页面，需要支持复杂的筛选和展示逻辑
- 任务单编号有特定的生成规则，前端需要按格式展示
- 二级列表固定展示3行作品-CP对应关系，超出需要滚动或分页
- 下架说明字段需要实现截断+悬停提示的交互

**前端设计**：

- **布局**：
  - 页面顶部：搜索筛选区域，采用多行表单布局
  - 页面中部：操作栏（创建任务单按钮 + 排序提示）
  - 页面主体：数据表格，支持展开/收起二级列表
  - 页面底部：分页组件

- **交互**：
  - 点击任务单编号跳转详情页
  - 点击行首展开箭头（▼/▲）展开/收起二级列表
  - 下架说明超出长度截断显示，鼠标悬停展示Tooltip完整内容
  - 审核拒绝状态悬停显示Tooltip展示审核意见

- **校验**：
  - 截止执行日期范围搜索：开始日期不能大于结束日期
  - 各搜索项支持组合筛选

- **修改位置**：
  - `views/video-manage/video-shelves/list.vue`：列表页主文件，实现搜索、表格、分页
  - 公共组件：可能需要封装展开表格组件

**验收标准**：

- ✓ 列表页正确展示所有任务单数据
- ✓ 任务单编号按V+YYMMDD+四位数字格式展示
- ✓ 搜索条件支持组合筛选，点击查询后列表刷新
- ✓ 点击展开箭头正确展示二级列表（作品-CP-频道信息）
- ✓ 下架说明超出长度截断，悬停显示完整内容
- ✓ 审核拒绝状态悬停显示审核意见Tooltip

### Quest 2: 视频下架列表页 - Tab切换与排序

**产品原文**：
> **Tab筛选与排序规则**：
> - 全部：展示所有状态的任务单，创建时间倒序（新→旧）
> - 创建中：创建时间倒序
> - 创建失败：创建时间倒序
> - 待审核：创建时间倒序
> - 审核拒绝：创建时间倒序
> - 待处理：按排队优先级排序（当天优先→日期→优先级→审批时间）
> - 处理中：按排队优先级排序
> - 已完成：完成时间倒序
>
> **处理中Tab排序规则**：按以下维度组合排序：
> 1. 是否当天截止：当天截止优先
> 2. 截止执行日期：升序（近→远）
> 3. 下架原因优先级：1>2>3>4>5>6（临时版权纠纷>调整上线时间>单作品解约>版权方解约>分销商解约>其他原因）
> 4. 审批通过时间：升序（先→后）
>
> **页面提示**：位置在操作栏右侧，文案"💡 当前按排队规则排序（当天截止优先，截止日期近→远，优先级高→低）"，说明ICON悬停显示Tooltip详细说明排序规则。

**原型图解读**：
参考原型图，Tab位于搜索区域上方，采用标准Tab切换样式。处理中/待处理Tab需要显示排序提示文案和说明Icon。

**需求解读**：

- Tab切换需要联动列表数据刷新
- 待处理/处理中Tab有特殊的排序逻辑，需要后端支持
- 需要展示排序规则提示，帮助用户理解列表排序逻辑

**前端设计**：

- **布局**：
  - Tab区域位于页面顶部，搜索区域上方
  - 操作栏右侧显示排序提示文案和说明Icon

- **交互**：
  - Tab切换时重新请求列表数据
  - 说明Icon悬停显示Tooltip，展示完整排序规则
  - 处理状态筛选下拉仅在"全部"Tab中展示

- **修改位置**：
  - `views/video-manage/video-shelves/list.vue`：Tab切换逻辑、排序提示展示

**验收标准**：

- ✓ Tab切换正确过滤列表数据
- ✓ 全部/创建中/创建失败/待审核/审核拒绝/已完成Tab按创建/完成时间倒序排序
- ✓ 待处理/处理中Tab按排队优先级排序
- ✓ 处理中/待处理Tab显示排序提示文案
- ✓ 点击说明Icon悬停显示排序规则Tooltip
- ✓ 处理状态筛选仅在"全部"Tab中可见

### Quest 3: 创建任务单抽屉 - 按作品创建

**产品原文**：
> **基础信息**：下架原因（下拉选择，必填）、下架说明（文本输入，上限100字符）、处理方式（单选按钮，视频删除/视频私享）、截止执行日期（日期选择，范围：当前日期至当前日期+6天）。
>
> **下架信息**：
> - 【+ 新增作品】：打开选择作品弹窗
> - 【批量导入】：打开批量导入弹窗
> - 已选作品表格字段：序号、作品名称（不可编辑）、CP名称（不可编辑）、注册频道ID（可编辑）、操作（删除）
>
> **添加作品弹窗**：弹窗标题"选择作品"，可选范围为当前AMS签约内容类型=视听作品、影视作品、图文作品、音乐作品。搜索条件：作品名称（模糊搜索）、CP名称（模糊搜索）。表格字段：作品名称、CP名称。选择方式：支持多选、一键全选。
>
> **批量导入弹窗**：提供下载导入模板链接，模板字段：作品名称、CP名称、注册频道ID。支持点击上传或拖拽文件上传，支持格式xls、xlsx，文件大小不超过5M。导入校验包括：模板格式、导入条数≤500条、作品名称必填、作品名称重复性、作品名称存在性、注册频道ID格式、作品与频道绑定关系、CP名称与作品匹配等。

**原型图解读**：
参考原型图，创建任务单采用抽屉（Drawer）形式从右侧滑出。表单分为基础信息和下架信息两部分。选择作品弹窗和批量导入弹窗采用Modal弹窗形式。

**需求解读**：

- 抽屉表单需要支持复杂的交互：添加作品、批量导入、表格编辑
- 作品选择弹窗需要支持搜索和多选
- 批量导入需要支持模板下载、文件上传、校验反馈
- 注册频道ID字段支持编辑，需要校验格式

**前端设计**：

- **布局**：
  - 抽屉从右侧滑出，宽度约800px
  - 抽屉内分区块：基础信息、下架信息
  - 下架信息区域：操作按钮（新增作品、批量导入）+ 已选作品表格

- **交互**：
  - 点击"新增作品"打开选择作品弹窗，支持搜索和多选
  - 点击"批量导入"打开导入弹窗，支持模板下载和文件上传
  - 已选作品表格支持删除行操作
  - 注册频道ID字段可编辑，支持多值输入
  - 底部按钮：取消、提交审批

- **校验**：
  - 下架原因、处理方式、截止执行日期必填
  - 截止执行日期范围：当前日期至+6天
  - 至少选择一个作品
  - 注册频道ID格式校验（YouTube频道ID规则）

- **修改位置**：
  - `views/video-manage/video-shelves/components/CreateTaskDrawer.vue`：创建任务单抽屉组件
  - 公共组件：作品选择弹窗、批量导入弹窗

**验收标准**：

- ✓ 抽屉正确滑出，表单字段正常展示
- ✓ 下架原因、处理方式、截止执行日期必填校验正常
- ✓ 点击"新增作品"打开弹窗，支持搜索和多选作品
- ✓ 选中作品后正确回显到已选作品表格
- ✓ 点击"批量导入"打开弹窗，支持模板下载
- ✓ 文件上传后正确校验并反馈结果
- ✓ 已选作品表格支持删除操作
- ✓ 注册频道ID支持编辑
- ✓ 提交审批时校验至少选择一个作品

### Quest 7: 创建解约单抽屉

**产品原文**：
> **基本信息**：分销商名称（下拉选择，必填）、解约类型（下拉选择，必填）、解约原因（文本输入，必填）、解约日期（日期选择，必填）、视频是否下架（下拉选择，是/否；解约类型=版权方解约/分销商解约时默认"是"且不可编辑；双方协商一致时默认"是"可编辑）。
>
> **视频下架信息**（视频是否下架=是时显示）：下架原因（下拉选择，必填；解约类型=版权方解约时默认"上游版权方解约"且不可编辑；解约类型=分销商解约时默认"分销商解约"且不可编辑；双方协商一致时无默认值）、下架说明（文本输入，上限100字符）、处理方式（单选按钮，默认"视频删除"，不可编辑）、截止执行日期（日期选择，范围：当前日期至+6天）。

> **解约信息**：提示"系统将根据作品所属分销商自动分组，生成多张解约单"，支持添加作品、批量导入，表格字段：作品名称、作品链接、作品规格、运作套餐、运作语种、操作（删除）。

> **提交逻辑**：视频是否下架=是时，提交前弹窗确认："请确认，基于此解约单创建「视频下架任务」吗？确认后将会自动发起「视频下架任务」审批"。确认后创建解约单并自动发起视频下架任务审批。
