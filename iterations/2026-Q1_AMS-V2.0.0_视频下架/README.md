## 数据流

```
AMS创建任务单 → 异步获取视频ID(通道关系) → 人工审核
  → 额度排队(总控系统查额度) → 执行下架(YouTube API)
  → 完成后推送 → 剧老板展示结果
```

```
AMS创建解约单(视频下架=是) → 自动创建视频下架任务单 → 同上流程
```
