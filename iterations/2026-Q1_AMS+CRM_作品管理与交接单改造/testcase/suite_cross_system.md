# 跨系统同步模块测试用例

**执行范围**：CRM→AMS 跨系统数据同步功能，包含创建提交同步、编辑重新交接同步、不可发布状态同步、同步失败处理、端到端全链路验证、同步延迟容忍验证、AMS本地设置与CRM推送并行验证、修改记录同步验证
**预估用例数**：12 条
**关联需求版本**：PRD V1.2 补充说明 §4
**预估执行时间**：约 40 分钟
**适用范围约束**：本期交接单改造仅支持线索类型=作品类（PGC），不支持频道类（UGC）。所有用例的测试数据须使用 PGC 类型的交接单/作品。

### ZWGL-SYNC-001 CRM创建交接单提交后AMS作品发布配置同步验证

**优先级**：P0
**用例类型**：功能测试
**所属模块**：跨系统同步
**关联需求**：PRD V1.2 补充说明 §4
**依赖用例**：无

**前置条件**：
1. [登录态][SYS-CRM] 使用账号 15057199668 / 验证码 1111 通过 SSO 登录，进入 CRM 系统
2. [登录态][SYS-AMS] 同一账号在另一标签页登录 AMS 系统（http://172.16.24.200:8024）
3. [页面定位][SYS-CRM] 导航到 CRM「交接单管理」列表页
4. [数据存在][数据准备: 环境预置] CRM 中已存在可选的目标作品 A（作品类线索/PGC 类型），且该作品在 AMS 列表中可搜索到
5. [数据存在][SYS-AMS] 在 AMS 作品列表中搜索作品 A，记录当前「内部是否可发布」列的值 {记录为:同步前发布状态}

**测试数据**：
| 字段 | 值 | 说明 |
|------|-----|------|
| 目标作品 | 作品 A | CRM 和 AMS 中均存在的作品 |
| 线索类型 | 作品类（PGC） | 本期仅支持 PGC 类型交接单 |
| 内部是否可发布 | 可发布 | |
| 首发时间 | 自定义时间 2026-06-01 | |
| 频道发布限制 | 仅CP指定频道 | |
| 字幕可发语种 | 选择 1 个语种 | |

**测试步骤**：
1. [SYS-CRM] 点击「新建交接单」按钮
2. [SYS-CRM] 向交接组添加作品 A
3. [SYS-CRM] 选择"可发布"，首发时间选"自定义时间"填 2026-06-01，频道限制选"仅CP指定频道"
4. [SYS-CRM] 勾选字幕可发语种
5. [SYS-CRM] 点击「提交」按钮
6. [SYS-CRM] 确认 Toast 提示 {包含:提交成功}
7. [SYS-AMS] 切换到 AMS 标签页
8. [SYS-AMS] 刷新作品列表页面
9. [SYS-AMS] 在作品列表中搜索作品 A
10. [SYS-AMS] 点击作品 A 名称打开详情抽屉

**预期结果**：
1. [SYS-CRM] 交接单提交成功，Toast {包含:提交成功}（L1）
2. [SYS-AMS] 作品 A 的「内部是否可发布」列显示蓝色圆点 + "可发布"（L1）
3. [SYS-AMS] 作品详情抽屉中「发布配置」卡片显示：
   - 内部可发布 = 可发布（L1）
   - 首发时间 = 自定义时间 2026-06-01（L1）
   - 可发频道 = 仅CP指定频道（L1）
4. [SYS-AMS] 作品详情抽屉中「语种配置」卡片可见，字幕可发语种显示已勾选的语种（L1）
5. [SYS-AMS] 页面无错误提示（L1 反向验证）

**是否需要人工介入**：否
**备注**：跨系统验证允许 5 秒数据同步延迟。首次验证失败时刷新页面重试一次。

### ZWGL-SYNC-002 CRM编辑已交接交接单重新交接后AMS数据更新验证

**优先级**：P0

**所属模块**：跨系统同步

**依赖用例**：ZWGL-SYNC-001（需先执行且通过，产出数据：一条已交接的交接单和 AMS 中已同步的作品 A）

1. [登录态][SYS-CRM] 已登录 CRM 系统
2. [登录态][SYS-AMS] 使用账号 15057199668 / 验证码 1111 通过 SSO 登录小五云平台，从应用中心进入「资产管理系统」（http://172.16.24.200:8024）
