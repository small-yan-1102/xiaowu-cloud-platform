**关联需求版本**：PRD V1.2 第四节 F03

### ZWGL-CREATE-001 新建交接单选择可发布完整填写并提交成功

**优先级**：P0
**用例类型**：冒烟测试
**所属模块**：CRM 交接单创建
**关联需求**：PRD V1.2 F03 §3.1-§3.7
**依赖用例**：无

**前置条件**：
1. [登录态][SYS-CRM] 使用账号 15057199668 / 验证码 1111 通过 SSO 登录小五云平台，从应用中心进入 CRM 系统，确认看到交接单管理页面
2. [页面定位] 导航到 CRM「交接单管理」列表页
3. [数据存在][数据准备: 环境预置] 系统中已存在可选的作品类线索（PGC）作品数据（创建交接单时作品搜索可返回结果）

**测试数据**：
| 字段 | 值 | 说明 |
|------|-----|------|
| 内部是否可发布 | 可发布 | 切换为可发布 |
| 首发时间类型 | 自定义时间 | 选择自定义 |
| 自定义日期 | 2026-06-01 | 未来日期 |
| 频道发布限制 | 无限制 | 原"合集" |
| 字幕可发语种 | 选择 1 个可用语种 | 从字典选取 |

**测试步骤**：
1. 点击「新建交接单」按钮
2. 在新建交接单弹窗中，向交接组添加一个作品（搜索并选择一个可用作品）
3. 在「内部是否可发布」区域选择"可发布"
4. 在「首发时间」区域选择"自定义时间"
5. 在日期选择器中选择 2026-06-01
6. 在「频道发布限制」区域选择"无限制"
7. 在字幕语种配置区域，勾选「字幕可发语种」中的 1 个语种
8. 点击「提交」按钮

**预期结果**：
1. 新建交接单弹窗出现，包含交接组配置区域（L1）
2. 作品已添加到交接组中，显示所选作品名称（L1）
3. 选择"可发布"后，发布配置区域（首发时间、频道发布限制、语种配置）可见（L1）
4. 首发时间区域显示"自定义时间"已选中，日期显示 2026-06-01（L1）
5. 频道发布限制显示"无限制"已选中（L1）
6. 字幕可发语种区域显示已勾选的语种（L1）
7. 点击提交后弹窗关闭（L1）
8. 页面顶部出现 Toast 提示 {包含:提交成功}（L1）
9. 交接单列表刷新，新增一条记录，状态为"已交接"（L1）
10. 页面无错误弹窗、无红色错误提示（L1 反向验证）

**是否需要人工介入**：否
**备注**：此用例产出一条"已交接"状态交接单，供编辑类用例和跨系统同步用例依赖。提交成功同时触发 CRM→AMS 数据同步。

### ZWGL-CREATE-022 首发时间自定义时间边界值验证

**优先级**：P2
**用例类型**：边界测试
**所属模块**：CRM 交接单创建
**关联需求**：PRD V1.2 F03 §3.6（首发时间 - 自定义时间选项）
**依赖用例**：无
**测试设计方法**：边界值分析 — CREATE-004 使用未来日期（有效值），本用例测试边界值（今天/过去日期）

**前置条件**：
1. [登录态][SYS-CRM] 使用账号 15057199668 / 验证码 1111 通过 SSO 登录小五云平台，从应用中心进入 CRM 系统，确认看到交接单管理页面
2. [页面定位] 导航到 CRM「交接单管理」列表页

**测试数据**：
| 字段 | 值 | 说明 |
|------|-----|------|
| 首发时间类型 | 自定义时间 | 选择自定义 |
| 自定义日期（边界1） | {当前日期} | 今天，边界值 |
| 自定义日期（边界2） | {当前日期-1天} | 昨天，过去日期边界值 |

**测试步骤**：
1. 点击「新建交接单」按钮
2. 在「内部是否可发布」区域选择"可发布"
3. 在「首发时间」区域选择"自定义时间"
4. 在日期选择器中选择今天的日期
5. 观察日期选择器是否允许选择今天日期，以及是否有校验提示
6. 在日期选择器中尝试选择昨天的日期
7. 观察日期选择器是否允许选择过去日期，以及是否有校验提示

**预期结果**：
1. 首发时间区域显示"自定义时间"已选中，日期选择器可见（L1）
2. 今天日期：系统允许选择，日期选择器正常接受今天的日期（L1）
3. 过去日期：PRD §3.6 明确规定「不限制范围」，系统应允许选择过去日期且无校验错误提示（L1）
4. 如过去日期被禁用（灰色/不可选）或选择后出现校验错误提示，标记为与 PRD 不一致的缺陷（L1）

**是否需要人工介入**：否
**备注**：CREATE-004 仅测试了未来日期（2026-06-01）的正常场景。本用例通过边界值分析方法，验证日期选择器对边界值（今天）和过去日期的处理。PRD V1.2 F03 §3.6 字段选项表明确首发时间自定义日期「不限制范围」，因此过去日期应可正常选择。
