### SMOKE-CRM-001 CRM创建PGC交接单（可发布+完整配置）提交成功

**优先级**：P0
**用例类型**：冒烟测试
**所属模块**：CRM 交接单创建
**关联需求**：PRD V1.2 F03 §3.1-§3.7
**关联PRJ用例**：ZWGL-CREATE-001
**预计执行时间**：~1 分钟
**依赖用例**：无

**前置条件**：
1. [登录态][SYS-CRM] 使用账号 15057199668 / 验证码 1111 通过 SSO 登录小五云平台，从应用中心进入 CRM 系统
2. [页面定位] 导航到 CRM「交接单管理」列表页
3. [数据存在][数据准备: 环境预置] 合约 D-YCZ0331002（CP: 西泠松舍）下的作品「完蛋，所有人都以为我是绝世高手」(ch_id=86442) 可选

**测试数据**：
| 字段 | 值 | 说明 |
|------|-----|------|
| 合约编号 | D-YCZ0331002 | CP: 西泠松舍 |
| 选择作品 | 完蛋，所有人都以为我是绝世高手 | CRM ch_id=86442, 视听作品 |
| 内部是否可发布 | 可发布 | 切换为可发布 |
| 首发时间 | 自定义时间 2026-06-01 | 未来日期 |
| 频道发布限制 | 无限制 | |
| 字幕可发语种 | 1 个语种 | 从字典选取 |

**测试步骤**：
1. 点击「新建交接单」按钮
2. 选择合约 D-YCZ0331002，在弹窗中向交接组添加作品「完蛋，所有人都以为我是绝世高手」（搜索并选择）
3. 选择"可发布"，首发时间选"自定义时间"填 2026-06-01，频道发布限制选"无限制"，勾选字幕可发语种 1 个
4. 点击「提交」按钮

**预期结果**：
1. 新建交接单弹窗出现，包含交接组配置区域（L1）
2. 选择"可发布"后发布配置区域（首发时间、频道发布限制、语种配置）展开可见（L1）
3. 点击提交后弹窗关闭，Toast {包含:提交成功}（L1）
4. 交接单列表新增一条记录，状态为"已交接"（L1）
5. 页面无错误弹窗、无红色错误提示（L1 反向验证）

**是否需要人工介入**：否
**备注**：此用例产出一条"已交接"状态交接单（合约 D-YCZ0331002 / 作品「完蛋，所有人都以为我是绝世高手」），供 SMOKE-CRM-002 和 SMOKE-AMS-001 依赖。提交成功同时触发 CRM→AMS 数据同步。

### SMOKE-CRM-002 CRM编辑已交接交接单修改发布配置并重新交接

**优先级**：P0
**用例类型**：冒烟测试
**所属模块**：CRM 交接单编辑
**关联需求**：PRD V1.2 F03-E §E.5
**关联PRJ用例**：ZWGL-EDIT-001
**预计执行时间**：~1 分钟
**依赖用例**：SMOKE-CRM-001（需先执行且通过，产出数据：一条状态为"已交接"的交接单）

**前置条件**：
1. [登录态][SYS-CRM] 使用账号 15057199668 / 验证码 1111 通过 SSO 登录，进入 CRM 系统
2. [页面定位] 导航到 CRM「交接单管理」列表页
3. [数据存在][数据准备: 依赖用例 SMOKE-CRM-001] 列表中存在一条状态为"已交接"的交接单

**测试步骤**：
1. 找到 SMOKE-CRM-001 创建的"已交接"状态交接单，点击「编辑」按钮
2. 在编辑弹窗中将「频道发布限制」从"无限制"修改为"仅CP指定频道"
3. 点击「保存并重新交接」按钮

**预期结果**：
1. 编辑弹窗出现，已有数据回显（作品、发布配置等字段已填充）（L1）
2. 底部按钮文案为「保存并重新交接」（因交接单状态为已交接）（L1）
3. 点击保存后弹窗关闭，Toast {包含:编辑成功，已重新交接}（L1）
4. 交接单列表该记录状态仍为"已交接"（L1）
5. 页面无错误弹窗、无红色错误提示（L1 反向验证）

**是否需要人工介入**：否
**备注**：重新交接成功同时触发 CRM→AMS 数据同步。

### SMOKE-E2E-001 全链路CRM创建→AMS同步→CRM编辑→AMS更新

**关联需求**：PRD V1.2 补充说明 §4, F03 §3.1-§3.7, F03-E §E.5, F01 §1.7-§1.8
**关联PRJ用例**：ZWGL-SYNC-006
**预计执行时间**：~2 分钟
**依赖用例**：无（独立完整链路，自建数据）

**前置条件**：
1. [登录态][SYS-CRM] 使用账号 15057199668 / 验证码 1111 通过 SSO 登录，进入 CRM 系统
2. [登录态][SYS-AMS] 同一账号在另一标签页登录 AMS 系统（http://172.16.24.200:8024）
3. [数据存在][数据准备: 环境预置] 合约 DM-YCZ0331001（CP: 西泠松舍）下的作品「一剑倾城」(ch_id=86437) 可选

**测试数据**：
| 字段 | 创建时值 | 编辑修改值 | 说明 |
|------|---------|----------|------|
| 合约编号 | DM-YCZ0331001 | — | CP: 西泠松舍 |
| 目标作品 | 一剑倾城 | — | CRM ch_id=86437, 视听作品 |
| 内部是否可发布 | 可发布 | 保持可发布 | |
| 首发时间 | 自定义时间 2026-08-01 | 随时可发布 | 编辑时修改 |
| 频道发布限制 | 仅CP指定频道 | 无限制 | 编辑时修改 |
| 字幕可发语种 | 1 个语种 | 增加 1 个语种 | 编辑时修改 |

**测试步骤**：
1. [SYS-CRM] 新建交接单，选择合约 DM-YCZ0331001，添加作品「一剑倾城」，选择"可发布"，首发时间=自定义时间 2026-08-01，频道限制=仅CP指定频道，勾选字幕可发语种 1 个，点击「提交」

4. [SYS-CRM] 编辑并重新交接成功（L1）
5. [SYS-AMS] 步骤 9：发布配置已更新 — 首发时间=随时可发布、可发频道=无限制（L1）
6. [SYS-AMS] 步骤 9：语种配置已更新 — 字幕可发语种显示 2 个语种（L1）
7. [SYS-AMS] 内部可发布状态保持"可发布"不变（L1）
8. 页面无错误弹窗、无红色错误提示（L1 反向验证）
