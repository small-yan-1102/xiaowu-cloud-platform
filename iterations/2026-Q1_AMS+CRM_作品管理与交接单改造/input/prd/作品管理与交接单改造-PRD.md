## 二、背景介绍

### 需求背景

为后续分发系统视频发布/制作环节的合规性校验，需从上至下增加发布配置相关信息在作品维度的留存，故对 CRM 交接单与 AMS 作品管理进行改造。

### 全局说明

- **内部可发布**：控制作品是否允许在内部平台发布，值为「可发布 / 不可发布」
- **发布配置**：包含首发时间、频道发布限制，统称「发布配置」。频道发布限制对应系统原「单开/合集」字段：「无限制」=原「合集」，「仅CP指定频道」=原「单开」
- **语种配置**：包含字幕可发语种、字幕禁发语种、配音可发语种、配音禁发语种
- **联动规则**：内部可发布 = 不可发布时，所有发布配置和语种配置不展示、不可填写
- **空值处理**：所有展示字段无数据时统一显示「-」，不显示空白
- AMS 详情页右侧「作品信息」卡片及「平台数据」卡片数据暂无接口，**前端 Mock 展示**，不影响功能上线

#### F03 交接单创建：发布配置联动

##### 3.1 适用范围

- 新建交接单弹窗

##### 3.2 提交校验规则

保存草稿和提交均需通过必填项校验（必填项不能为空），两者校验规则一致。

**校验失败交互**：错误字段内联高亮提示，弹窗不关闭。

**保存草稿成功**：交接单状态设为「待交接」，Toast 「保存成功」，关闭弹窗。

**提交成功**：交接单状态设为「已交接」，Toast 「提交成功」，关闭弹窗，刷新列表，同时将发布配置和语种配置同步到 AMS 作品维度。

##### 3.3 联动规则

```
内部是否可发布 = 可发布  →  展示并可填写：频道发布限制、首发时间、字幕语种、配音语种
内部是否可发布 = 不可发布 →  隐藏：频道发布限制、首发时间、字幕语种、配音语种
```

切换「不可发布」前弹出确认提示：「切换后将清空发布配置和语种配置，是否继续？」 → 「确认」清空 / 「取消」留在当前选项。

**语种互斥规则：**
- 同一语种不能同时出现在「字幕可发」和「字幕禁发」中；选择某语种时自动从另一侧移除该语种。配音可发与配音禁发同理。

**语种选项来源**：与线上交接单语种字典保持一致，交互方式不变。字幕和配音语种，可发和禁发，至少选择一项。

##### 3.4 默认状态

- 新建弹窗默认选「不可发布」，发布配置区域默认隐藏

##### 3.5 切换为「不可发布」时自动重置

- 首发时间恢复「随时可发布」，清空自定义日期值
- 频道发布限制恢复「无限制」
- 字幕语种/配音语种所有勾选项取消勾选

##### 3.6 字段选项

| 字段 | 选项 | 备注 |
| :--- | :--- | :--- |
| 内部是否可发布 | 可发布 / 不可发布 | 必填 |
| 首发时间 | 随时可发布 / 自定义时间 / 暂不可发布，等待上线通知 | 内部可发布时必填；选「自定义时间」时日期不能为空，不限制范围 |
| 频道发布限制 | 无限制（原「合集」）/ 仅CP指定频道（原「单开」） | 内部可发布时必填 |
| 字幕语种 / 配音语种 | 选填，语种列表与线上一致（字典数据） | 可不填；同一语种不能同时出现在可发和禁发 |

##### 3.7 「同步到配音」快捷操作

**位置**：语种配置区块内，「字幕禁发语种」下方、「配音可发语种」上方，附带说明文案「将字幕可发/禁发同步至配音」。

**功能**：点击后将「字幕可发语种」和「字幕禁发语种」的当前勾选值一键同步覆盖至「配音可发语种」和「配音禁发语种」，减少重复填写。

| 规则项 | 说明 |
| :--- | :--- |
| 触发方式 | 点击「同步到配音」按钮 |
| 同步行为 | 字幕可发 → 覆盖写入配音可发；字幕禁发 → 覆盖写入配音禁发 |
| 是否弹窗 | 否，即时在表单内更新，无弹窗、无加载动画 |
| 同步后是否可修改 | 是，同步后配音字段仍可手动再次调整 |
| 字幕语种为空时 | 同步后配音对应字段变为空（清空已选值） |

#### F03-E 交接单编辑

##### E.1 操作权限

- **可编辑状态**：交接单**任何状态下均可编辑**

##### E.2 弹窗规格

- 弹窗标题：「编辑交接单」
- 宽度、表单结构与「创建交接单」弹窗保持一致
- **编辑弹窗不支持新增/删除交接组**，交接组数量固定为原交接单的交接组数量
- 底部操作按钮：「取消」+「保存并重新交接」

##### E.3 数据回显

打开编辑弹窗时，**自动将当前交接单已有数据回填至对应字段**：

| 字段 | 回显说明 |
| :--- | :--- |
| 交接组/作品列表 | 按已有交接组数量渲染，每组内作品及字段值回填 |
| 内部是否可发布 | 回显已选值，并据此触发发布配置区域的显示/隐藏 |
| 首发时间 | 回显已选类型及自定义日期（如有） |
| 频道发布限制 | 回显已选值 |
| 字幕可发语种 | 回显已勾选的语种 |
| 字幕禁发语种 | 回显已勾选的语种 |
| 配音可发语种 | 回显已勾选的语种 |
| 配音禁发语种 | 回显已勾选的语种 |
| YT账号归属方 | 回显已选值 |
| 附件 | 回显已上传/填写的附件链接 |
| 特殊运作要求 | 回显已填写内容 |
| 对播放量/更新频次/收益要求 | 回显已填写内容 |

回显完成后，所有字段均可修改。

##### E.4 发布配置联动（与创建弹窗一致）

```
内部是否可发布 = 可发布  →  展示并可修改：频道发布限制、首发时间、字幕语种、配音语种
内部是否可发布 = 不可发布 →  隐藏并清空：频道发布限制、首发时间、字幕语种、配音语种
```

切换「不可发布」前弹出确认提示：「切换后将清空发布配置和语种配置，是否继续？」 → 「确认」清空 / 「取消」留在当前选项。

##### E.4.1 「同步到配音」快捷操作（与创建弹窗一致）

编辑弹窗中语种配置区块同样提供「同步到配音」按钮，位置与交互规则与 F03 § 3.6 完全一致，此处不再赘述。

##### E.5 保存并重新交接

点击「保存并重新交接」：

1. 执行表单校验（必填项不能为空）
2. 校验通过后，保存编辑内容并**触发再次交接**（相当于重新提交一次交接流程，仅交接单状态=已交接时，触发再次交接）
3. 操作成功：弹窗关闭，以下两种情况分别提示：
   - **交接单状态 = 已交接**：Toast 「编辑成功，已重新交接」，刷新列表，同时将发布配置和语种配置同步到 AMS 作品维度
   - **其他状态**：按钮文案显示为「保存」，Toast 「编辑成功」，刷新列表
4. 操作失败：保持弹窗，Toast 提示「操作失败，{原因}」

##### E.6 关闭弹窗时的提示

如用户对表单内容有改动后点击「取消」或关闭按钮，弹出确认提示：

> 「有未保存的修改，确认关闭？」→ 「确认」关闭 / 「取消」留在弹窗

### 补充说明

1. **联动交互响应**：CRM 创建/编辑弹窗，发布配置联动（显示/隐藏）即时响应，无需刷新页面
2. **空值统一**：所有详情展示字段无数据时统一显示「-」，不显示空白
3. **操作反馈**：
   - 关闭抽屉时无需确认（只读，无数据修改）
4. **CRM→AMS 数据同步规则**：
   - **触发时机**：交接单「提交」成功，或已交接状态交接单「保存并重新交接」成功时，自动将该交接单内每个作品的发布配置和语种配置写入对应 AMS 作品维度。
   - **同步单元**：以作品为单位，交接组概念不影响同步，每个作品独立同步其发布配置。
   - **同步失败**：Toast 提示「同步失败，请重试」，列表刷新后 AMS 展示将为旧数据。
5. **本期不涉及通知机制**：交接单提交/重新交接后，本期不触发任何站内信/邮件/消息通知。

   - 详情抽屉从右侧滑入，动画流畅
   - 发布配置联动（显示/隐藏）即时响应，无延迟感
   - 左右两栏各自独立滚动，互不影响

3. **数据一致性**：切换为「不可发布」时，须主动清空关联字段值，避免提交脏数据

4. **空值处理**：所有展示字段无数据时统一显示「-」，不显示空白

5. **播放量格式化**：≥1亿显示X.X亿；≥1万显示X.X万；其余原样显示

## 六、历史数据订正方案

> 本次改造新增多个字段，历史交接单在这些字段上均无存量数据，需根据以下规则对存量交接单进行一次性批量订正。**执行时机：上线后由开发执行，具体策略由开发制订。**

### 6.1 首发时间

| 原始数据状态 | 订正目标字段 | 订正值 |
| :--- | :--- | :--- |
| 首发时间字段有具体日期 | publishTimeType | `custom`，保留原 publishFirstTime 日期值不变 |
| 首发时间字段为空 / null | publishTimeType | `anytime`，写入【随时可发布】 |

### 6.2 频道发布限制

> 对应系统原字段「单开/合集」

| 原字段名 | 原枚举值 | 订正目标字段 | 订正值 |
| :--- | :--- | :--- | :--- |
| 单开/合集 | `合集` | channelLimit | `unlimited`（无限制） |
| 单开/合集 | `单开` | channelLimit | `cp_only`（仅CP指定频道） |
| 单开/合集 | 为空 / null | channelLimit | `unlimited`（无限制） |

### 6.3 内部是否可发布

存量交接单全部初始化为【可发布】，即 internalPublish = `true`。

### 6.4 语种配置

| 原字段名 | 订正目标字段 | 说明 |
| :--- | :--- | :--- |
| 发布语种 | subtitleLangs（字幕可发）+ dubbingLangs（配音可发） | 将原「发布语种」字段的内容同时写入两个字段 |
| 禁发语种 | subtitleForbidLangs（字幕禁发）+ dubbingForbidLangs（配音禁发） | 将原「禁发语种」字段的内容同时写入两个字段 |
