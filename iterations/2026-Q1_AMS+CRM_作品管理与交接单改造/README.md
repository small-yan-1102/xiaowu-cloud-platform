## 数据流

```
CRM 交接单创建/编辑(录入发布配置+语种配置)
  → 交接至 AMS(数据同步)
  → AMS 作品详情页展示(概览/修改记录)
  → AMS 设置弹窗可编辑(发布配置/语种配置)
  → 下游分发系统消费(合规性校验)
```
